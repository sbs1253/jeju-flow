# 🌴 Jeju Flow: 데이터 기반 제주 문화 트렌드 대시보드

제주 지역의 **수요(네이버 검색 트렌드)**와 **공급(KOPIS 공연 현황)** 데이터를 융합하여, AI가 실시간 문화 인사이트를 제안하는 하이엔드 에디토리얼 대시보드 프로젝트입니다.

---

## 1. 프로젝트 개요 (Project Overview)

### 🎯 기획 의도
제주도는 대한민국 최고의 관광지이자 문화 예술의 거점이지만, 실제 방문객들이 원하는 '문화적 요구'와 현장에서 제공되는 '공연/축제 공급' 사이에는 정보의 비대칭이 존재합니다. 
**Jeju Flow**는 이러한 격차를 데이터로 시각화하고, AI(Gemini)를 통해 전문가 수준의 문화 기획 아이디어를 자동으로 도출하여 제주 문화 산업의 의사결정을 돕기 위해 기획되었습니다.

### 👥 타겟 사용자 (User Targeting)
1. **문화 행사 기획자**: 데이터에 기반한 새로운 축제 및 전시 테마 기획이 필요한 전문가
2. **지자체 관계자**: 지역별 문화 시설 활용도 및 트렌드 분석이 필요한 정책 담당자
3. **데이터 마케터**: 제주도 방문객의 관심사 변화를 실시간으로 추적하고자 하는 마케터
4. **문화 예술 애호가**: 제주에서 열리는 양질의 공연 정보를 트렌드와 함께 확인하고 싶은 사용자

### 👤 페르소나 (User Persona)
- **성함**: 김제주 (34세, 문화 기획자)
- **고민**: "매년 반복되는 축제 대신, 요즘 사람들이 정말 관심을 가지는 키워드로 새로운 전시를 기획하고 싶어. 근데 일일이 검색량을 비교하고 공연 현황을 파악하기엔 시간이 너무 부족해."
- **해결**: Jeju Flow의 'AI 인사이트' 기능을 통해 이번 주 급상승 키워드와 연관된 공연 기획안을 단 5초 만에 제안받고, 실제 공급 현황(KOPIS)과 비교하여 실행 가능성을 검토함.

---

## 2. 주요 기능 (Core Features)

### 📈 실시간 검색 트렌드 분석 (Naver DataLab 연동)
- 네이버 데이터랩 API를 통해 4가지 핵심 카테고리(전체, 전시, 뮤지컬, 축제)의 검색량을 실시간 수집.
- 최근 30일 vs 이전 30일 데이터를 비교하여 트렌드 변화율(Change Rate) 자동 산출.

### 🎭 공연 및 축제 심층 데이터 동기화 (KOPIS Deep Sync)
- KOPIS API를 통해 제주 지역 공연의 상세 정보(포스터, 가격, 캐스팅, 상세 소개 이미지)를 전수 수집.
- 단순히 목록만 보여주는 것이 아니라, 데이터베이스에 상세 정보를 아카이빙하여 빠른 검색 및 분석 제공.

### 🤖 Gemini AI 통합 인사이트 (AI-Powered Planning)
- **수요-공급 격차 분석**: 검색량은 많지만 공연 공급이 부족한 분야를 AI가 자동으로 포착.
- **창의적 기획 제안**: 수집된 트렌드 키워드를 바탕으로 실제 실행 가능한 6~8개의 문화 기획 아이디어 생성.
- **주간 스냅샷**: 분석된 인사이트를 DB에 저장하여 과거 트렌드 흐름을 추적할 수 있는 아카이빙 기능.

### ⚡ 자동화 파이프라인 (GitHub Actions)
- 매일 아침 6시(KST), GitHub Actions가 자동으로 데이터 수집 및 AI 분석 엔드포인트를 호출하여 최신성을 유지.

## 3. 시스템 아키텍처 (System Architecture)

### 🏗️ 기술 스택 (Tech Stack)
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4, shadcn/ui, Framer Motion (고급 애니메이션)
- **Database**: Supabase (PostgreSQL)
- **AI**: Google Gemini 1.5 Flash / 3.1 Flash Lite
- **Visualization**: Recharts (인터랙티브 차트)
- **Automation**: GitHub Actions (Cron Jobs)
- **Deployment**: Vercel

---

## 4. 데이터 구조 (Data Schema)

- **search_trends**: 네이버 검색 비율 및 키워드 그룹 관리
- **performances**: 공연 상세 정보, 가격, 이미지, 지역 데이터 (Upsert 방식)
- **ai_insights**: AI가 생성한 주간 요약, 기획 아이디어, 키워드 분석 결과 저장

---

## 5. 프로젝트의 차별점

1. **에디토리얼 디자인**: 단순한 관리자 페이지가 아닌, 잡지나 미디어 아카이브를 보는 듯한 유려한 UI/UX.
2. **실시간성**: 매일 자동으로 업데이트되는 데이터 수집 및 분석 자동화 체계.
3. **실무 지향적 AI**: 단순 요약이 아닌, '기획자'의 관점에서 구체적인 타겟과 실행 시기를 제안하는 프롬프트 엔지니어링.
